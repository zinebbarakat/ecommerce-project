About Zaza’s Tech House

A simple ecommerce platform for browsing products, managing a cart, and tracking orders.

Our Mission

Zaza’s Tech House is built to keep online shopping simple: a clean catalog, clear product details, and an easy checkout flow. The interface is designed to work well on desktop and mobile.

What you can do

  • Browse products and filter by category
  • Open product pages to view price, stock, and descriptions
  • Add items to your cart and update quantities
  • Checkout and track confirmed orders

How it works (quick overview)

The frontend communicates with a REST API built with Express. Data is stored in SQLite (users, products, orders, and order items).

The cart is implemented as an order with status CART. When the admin confirms it, it becomes a confirmed order with status ORDER.

Project Stack

  • Frontend: HTML, CSS, JavaScript (modules)
  • Backend: Node.js + Express
  • Database: SQLite
  • Auth: Simple session stored in localStorage + headers
